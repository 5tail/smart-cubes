事件 Log

    輸入方塊 MAC address

    GAN 方塊需要藍牙 MAC 才能推導金鑰。若瀏覽器沒開自動偵測，請手動輸入一次 （之後會記住這顆方塊，不再詢問）。

    MAC 在哪裡找？
    • Windows：裝置管理員 → 藍牙 → 方塊 → 內容 → 詳細資料 → 選「Bluetooth 裝置位址」。
    • macOS：關於這台 Mac → 系統報告 → 藍牙 → 找到方塊 → 看 Address。
    • 或 GAN 官方 App「Cube Station」裡也看得到。

    Phase 2：透過 connectSmartCube()（GAN）、connectQiyiCube()、 connectMoyuCube() 連線對應方塊，即時顯示轉動記錄、方塊當前狀態與電量。 需要桌機 Chrome / Edge 或 Android Chrome（Web Bluetooth）。
