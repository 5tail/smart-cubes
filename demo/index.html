事件 Log

    輸入方塊 MAC address

    這顆方塊需要藍牙 MAC 才能完成連線（GAN 用於推導金鑰；QiYi 的 hello 必須帶對 MAC 方塊才回話）。兩條路擇一：開啟瀏覽器自動偵測 （推薦，設定一次、之後免輸入），或往下手動輸入一次 （會記住這顆方塊，不再詢問）。

    ⚡ 免輸入模式：讓 Chrome 自動抓 MAC（設定一次就好）

    1. 複製這個網址（chrome:// 網址不能用點的，要貼到網址列）： chrome://flags/#enable-experimental-web-platform-features
    2. 貼到 Chrome 網址列開啟，把「Experimental Web Platform features」設為 Enabled
    3. 按 Relaunch 重啟瀏覽器
    4. 回到本頁重新連線 —— 之後連 GAN 都不用輸入 MAC

    電腦與 Android 平板的 Chrome 都適用。此旗標只是開啟「網頁讀取藍牙廣播」的 實驗功能（方塊 MAC 藏在廣播裡），不影響其他網站。

    MAC 在哪裡找？
    • Windows：裝置管理員 → 藍牙 → 方塊 → 內容 → 詳細資料 → 選「Bluetooth 裝置位址」。
    • macOS：關於這台 Mac → 系統報告 → 藍牙 → 找到方塊 → 看 Address。
    • 或 GAN 官方 App「Cube Station」裡也看得到。

    支援品牌與已知限制

    品牌型號狀態
    GAN GAN12 UI / i4 / i Carry（Gen2/3/4） ✅ 實機驗收通過
    MoYu 魔域 WeiLong AI（WCU_MY32） ✅ 實機驗收通過
    QiYi 奇藝 QiYi AI 3x3（QY-QYSC）、Tornado V4-i（XMD-TornadoV4-i） ✅ 實機驗收通過
    QiYi 奇藝 Tornado V4 LE（XMD-TornadoV4LE） ✅ 實機驗收通過
    • 需要瀏覽器支援 Web Bluetooth：桌機 Chrome / Edge 或 Android Chrome； iOS Safari 與 Firefox 不支援。
    • GAN / QiYi 需要 MAC：開 chrome://flags/#enable-experimental-web-platform-features 讓瀏覽器自動抓， 或手動輸入一次（會記住這顆方塊）。QiYi 通常可由名稱自動推導，推導不對時才會詢問。
    • QiYi 三型號（QY-QYSC / Tornado V4-i / V4 LE）皆實機驗收通過，同一組協議。
    • 手上型號沒列到？用上方「🔍 診斷方塊 / 🔴 錄製封包」擷取封包，依 CONTRIBUTING.md 回傳即可協助擴充。
